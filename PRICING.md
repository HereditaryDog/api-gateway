# API Gateway 计费方案

> 基于各大厂商官方定价，加上合理利润率

---

## 💰 定价策略

### 利润率设置
- **基础加成**: 成本价 × 1.3 (30% 利润)
- **热门模型**: 成本价 × 1.5 (50% 利润)
- **特价促销**: 成本价 × 1.2 (20% 利润)

### 余额计算
- 1 USD = 1 平台余额
- 用户充值按 USD 计算

## 🔄 计费方式

### 实时计费
```
实际费用 = (Input Tokens × Input 单价 + Output Tokens × Output 单价) / 1,000,000
```

### 计费示例
**示例 1**: 调用 gpt-4o
- Input: 1000 tokens × $3.25 / 1M = $0.00325
- Output: 500 tokens × $13.00 / 1M = $0.0065
- 总计: $0.00975

**示例 2**: 调用 claude-3-opus
- Input: 2000 tokens × $19.50 / 1M = $0.039
- Output: 1000 tokens × $97.50 / 1M = $0.0975
- 总计: $0.1365

## 🏷️ 定价建议

### 引流策略
- **GLM-4-Flash**: 超低价，吸引用户注册
- **硅基流动**: 免费/几乎免费，作为试用

### 利润模型
- **走量模型**: gpt-4o-mini, gpt-3.5-turbo (量大利润高)
- **高端模型**: gpt-4, claude-3-opus (单价高利润高)
- **国产替代**: DeepSeek, 智谱 (价格优势明显)

## 📝 配置文件

定价配置保存在 `config/pricing.yaml`:

```yaml
pricing:
  default_multiplier: 1.3  # 默认利润率
  
  models:
    # OpenAI
    "gpt-4o":
      input: 3.25   # $/1M tokens
      output: 13.00
    
    "gpt-4o-mini":
      input: 0.20
      output: 0.80
    
    # Anthropic
    "claude-3-opus":
      input: 19.50
      output: 97.50
    
    # DeepSeek
    "deepseek-chat":
      input: 0.20
      output: 0.40
```
